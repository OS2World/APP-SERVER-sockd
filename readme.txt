A simple SOCKS server for OS/2
==============================

This package contains a very simple SOCKS server for OS/2. It supports
version 4 of the SOCKS protocol.

The package contains the following files:

README.TXT	- this file.
SOCKD.DOC	- basic documentation.
SOCKD.EXE	- the program executable.
SOCKD.CNF	- sample SOCKD configuration file
NETLIB.DLL	- a library of functions used by this and related programs.

That's all!
